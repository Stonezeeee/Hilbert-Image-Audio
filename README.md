可以把图片和音频使用希尔伯特曲线算法相互转换

使用到的工具库：

numpy

hilbertcurve:希尔伯特曲线函数

pillow:图片处理

scipy:wav格式音频生成

pydub:mp3格式转wav

    此库需要使用FFmpeg,可前往 https://www.gyan.dev/ffmpeg/builds/ 下载,并将bin文件夹的路径添加到系统环境变量
    
    若不想使用mp3转wav功能可以不引用该库，仅在mp3_to_wav()函数中被使用
    
代码文件中包含四个功能函数和一个主函数，函数功能和参数已经标注在其内部，主函数中有一份示例代码

音频长度(单位s) = 图片像素个数(分辨率的平方) * 每像素采样数 / 44100

图片分辨率 = 2^希尔伯特曲线的阶数

更详细的介绍见 [使用希尔伯特曲线，我把一首歌扔到了图片里！！！](https://www.bilibili.com/opus/1160131092607401986)
